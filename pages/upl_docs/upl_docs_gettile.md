## GetTile

Retrieve the internal id of a tile at a certain position. The internal id corresponds to the id you see when hovering over a tile in LDTK.

### Inputs:
- Numeric: id of the layer
- Numeric: x position
- Numeric: y position

### Outputs:
- Numeric: internal id of the tile
